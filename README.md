# weather
这是我学习android的第一个项目，在程序的里面可以看到第一行代码的痕迹，没错，我就是在coolWeather的基础上进行第二次开发的，界面模仿苹果的天气（虽然一点都不像），在项目里面用到的框架有OKHttp,eventbus,gson,高德地图的定位。所以可以看得出来这是一个小白的学习作品，如果你也是一个android的初学者，我相信你能从中找到自信，愿我们在这条路上能走下去。<br>
